像 Any-Block 就内置了很多这种功能，例如列表转化为其他树类格式，详见 [列表选择器](1.%20使用示例%20-%20列表选择器.md)。

### 借用代码格式

反过来也行，例如mermaid是没有提供代码高亮功能的，写起来比较痛苦。
但在这里我们可以借助js的**高亮**来写mermaid代码，然后渲染回mermaid格式
[Xcode]
```js
~~~mermaid
stateDiagram
    [*] --> Still
    Still --> [*]

    Still --> Moving
    Moving --> Still
    Moving --> Crash
    Crash --> [*]
~~~
```
