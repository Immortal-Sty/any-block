# mermaid类

这部分是扩展处理器，用了mermaid库

## title2mindmap

```js
// 纯组合，后续用别名模块替代
id: "title2mindmap",
name: "标题到脑图",
process_param: ABConvert_IOEnum.text,
process_return: ABConvert_IOEnum.el,
```

## list2mindmap

```js
// 纯组合，后续用别名模块替代
id: "list2mindmap",
name: "列表转mermaid思维导图",
process_param: ABConvert_IOEnum.text,
process_return: ABConvert_IOEnum.el,
```

## list2mermaid

```js
id: "list2mermaid",

```

## list2mehrmaidText

id: "list2mehrmaidText",
